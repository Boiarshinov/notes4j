## Виды интеграции

- Интеграция через БД - сервисы одновременно пользуются общей БД.
- Remote Procedure Call - вызов методов удаленного сервиса (в Java это называется RMI)
- CORBA (в Java называется EJB)
- SOAP
- [REST-сервисы](../web/rest.md)
