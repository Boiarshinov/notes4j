# Процедурные расширения

Процедурные расширения делятся на:

- Триггеры
- Хранимые процедуры

В различных СУБД можно писать процедурные расширения на различных высокоуровневых языках

|     |     |
| --- | --- |
| **СУБД** | **Язык** |
| MySQL | Java |
| Oracle | Java |
| MSSQL | C#  |
| PostgreSQL | Python |
