# Терминология

- **DBMS** - data base modification system - СУБД
- **DDL SQL -** data definition language - язык описания структуры базы данных
- **DML SQL** - data modification language - совокупность команд, отвечающих за внесение изменений в содержимое баз данных
- **ORM** - Object Relationing Mapping - технология программирования, которая связывает базы данных с концепциями ООП ЯП, создавая виртуальную объектную базу данных
- **RAID** - Redundant Array of Independent Disks - избыточный массив независимых дисков
- **SQL** - Structured Query Language
- **Атрибут** - Свойства, характеризующие сущность - название столбца в таблице
- **БД** - база данных
- **Домен** - набор допустимых значений атрибутов
- **Идентификатор** - атрибут с уникальным значением для данной таблицы
- **Ключ** - минимальный набор атрибутов однозначно определяющий уникальный кортеж
- **Кортеж** - строка в таблице, представляющая собой набор значений различных атрибутов
- **Объект** - экземпляр сущности
- **СУБД** - система управления базами данных
- **Суррогатный ключ** - специально добавленный ключ
- **Сущность** - класс, хранящийся в БД. Другими словами - таблица
- **Схема** - база данных
- **Транзакция** - группа операций, которая представляет собой одно законченное действие. Транзакция либо выполняется полностью, либо не выполняется вообще.
